# Habit-Tracker

Habit Tracker

- A simple and responsive habit-tracking web application built with HTML, CSS, and vanilla JavaScript. It includes a dark/light theme toggle, custom icons, a modal for creating habits, and full localStorage support. Everything runs in the browserâ€”no backend required.

Features

   - Add new habits with a title and icon
   - Mark habits as complete or incomplete
   - Delete habits using a right-click context menu    
   - Persistent dark/light theme toggle    
   - All data stored in localStorage    
   - Responsive grid layout  
   - Smooth and accessible modal interface

Tech Stack

   - HTML5
   - CSS3 (CSS Grid, custom properties, responsive design)
   -JavaScript (DOM manipulation, localStorage, modular structure)

Project Structure

    -main.css
    -main.js
